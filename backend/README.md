# AI Resume Screening & Recruiter Evaluation Platform

This project is a full-stack web application built to simplify the resume screening process for recruiters.  
It helps in automatically analyzing resumes, identifying candidate skills, calculating scores, and managing hiring decisions through an easy-to-use dashboard.

The idea behind this project was to simulate how modern companies use Applicant Tracking Systems (ATS) to shortlist candidates efficiently.

---

## 🚀 What This Project Does

Instead of manually reading every resume, recruiters can:

- Upload candidate resumes (PDF)
- Automatically detect technical skills
- Generate a candidate score
- View all applicants in a dashboard
- Search candidates by skills
- Shortlist or reject applicants
- View detailed candidate profiles
- Read resumes directly inside the platform
- Add recruiter feedback and notes

## ⚙️ How It Works

1. Recruiter uploads a resume.
2. Backend extracts text from the PDF.
3. Skills are detected using predefined logic.
4. A score is generated automatically.
5. Candidate appears in recruiter dashboard.
6. Recruiter reviews, adds notes, and makes hiring decisions.
