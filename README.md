## 🤖 AI Resume Screening & Candidate Evaluation Platform

An AI-powered platform that helps recruiters automatically analyze resumes, extract skills, and evaluate candidates efficiently.

## ✨ Features

- 📄 Upload resumes in PDF format
- 🤖 AI-powered resume analysis
- 🎯 Skill extraction from resumes
- 📊 Candidate scoring based on job requirements
- 🔍 Smart filtering of candidates
- ⚡ Fast and automated screening process
